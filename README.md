## вңҰ Sobre el Proyecto
 
**ExUp Energy** es una aplicaciГіn Android construida con Flutter que permite a los usuarios localizar gasolineras en tiempo real, comparar precios de combustible y detectar automГЎticamente oportunidades de ahorro en su zona geogrГЎfica.
 
El proyecto nace con un objetivo tГ©cnico claro: aplicar estГЎndares de ingenierГӯa de software de nivel profesional en un entorno mГіvil multiplataforma, integrando datos geoespaciales en tiempo real con una arquitectura limpia, testeable y mantenible a largo plazo.

### BLoC Pattern вҖ” GestiГіn de Estado
 
Cada feature encapsula su propio BLoC o Cubit, siguiendo el flujo unidireccional de datos:
 
```
UI Event в”Җв”Җв–ә BLoC в”Җв”Җв–ә Use Case в”Җв”Җв–ә Repository в”Җв”Җв–ә Data Source
                в”Ӯ
                в–ј
           State Stream в”Җв”Җв–ә UI Rebuild
```
 
Este patrГіn asegura que la interfaz sea un reflejo puro del estado, sin lГіgica de negocio embebida en los widgets.

El pipeline de entrega estГЎ completamente automatizado mediante **Gitea Actions** en un entorno Dockerizado:
 

| Etapa | Herramienta | DescripciГіn |
